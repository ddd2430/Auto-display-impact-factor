@By ddd

共三个个版本，不要放置在含有中文的路径下。

离线版（offline）比较稳定，依赖ddd.xlsx为数据来源，所显示的最新影响因子指的是2016-2017，如果要查找更新的，请更新ddd.xlsx,并按ddd.xlsx的顺序排列各列数据。

在线版1（online1）打开速度和查找速度都比较慢，请耐心等待，数据来源为Letpub网站，依赖phantomjs，请勿删除或重命名该文件夹及其中的文件。

在线版2（online2）是通过数据发送和接收分析完成查询的，与online1相比，不需要phantomjs的支持，程序（online2.exe）可以单独运行，而且查询速度稍有提高，对内存资源的消耗极少。

源码需要python2.7打开，依赖相关的第三方库。
